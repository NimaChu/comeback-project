# Comeback

Comeback 是一个面向移动端场景的 AI 话术生成器应用包，适合上传到 Nima Tech Space 这类静态应用市场。

它的目标很简单：在社交、职场、关系沟通等语境里，帮助用户更快生成自然、得体、可直接使用的回复。

## Features

- 多场景话术生成
- 多种表达风格切换
- 面向移动端的轻量交互
- 使用平台统一模型接口，不在前端内置第三方密钥
- 已整理成可直接上传的平台应用包结构

## Package Structure

```text
manifest.json
README.md
assets/
app/
```

- `manifest.json`: 应用元数据
- `assets/`: 图标、封面等静态资源
- `app/`: 可直接运行的静态前端文件

## Usage

上传到平台后，用户可以：

1. 选择场景和风格
2. 输入原始消息或自己想表达的内容
3. 一键生成回复
4. 复制并直接发送

## Upload Notes

这个目录本身就是一个完整的应用包目录，压缩成 zip 后即可上传。

上传时建议：

- 使用唯一 slug，避免与其他作者冲突
- 如果应用需要 AI，上传时选择合适的模型类型
- 保持前端资源为相对路径，避免上传后静态资源失效
